TaskFlow – Smart To-Do List

A Simple Yet Effective Task Management App

🚀 Overview

TaskFlow is a clean and interactive To-Do List web application designed to help users manage their daily tasks efficiently.

Built using HTML, CSS, and JavaScript, this project focuses on simplicity, usability, and core functionality — while also introducing persistent storage using the browser.

✨ Features
➕ Add Tasks
Easily add new tasks using an input field and button

✔️ Mark as Complete
Click on any task to toggle its completion status (with strikethrough effect)

❌ Delete Tasks
Remove tasks instantly using a delete (×) button

💾 Persistent Storage
Tasks are stored using localStorage, so your data remains even after refreshing or closing the browser

🎨 Clean UI Design
Minimal, user-friendly interface with icons for better experience

🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling and layout
JavaScript – Logic and interactivity
localStorage – Data persistence
